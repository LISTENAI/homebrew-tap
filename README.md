# listenai/homebrew-tap

[Homebrew](https://brew.sh) tap for [LISTENAI](https://github.com/LISTENAI) tools.

## Usage

```sh
brew tap listenai/tap
```

Then install any formula or cask it provides:

```sh
brew install listenai/tap/<name>          # command-line tools
brew install --cask listenai/tap/<name>   # apps
```

Browse what's available with `brew search listenai/tap`. Once tapped, you can also
drop the `listenai/tap/` prefix (e.g. `brew install <name>`).
